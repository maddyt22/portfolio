Lisa Bennett Studio

Website Design and Development

Brief

Create a website for a local artist to display her artworks, sell pieces and advertise upcoming events.

Creative Direction

Working along side another designer, we drafted some wireframes for the website. After client approval we got to work with building the website in Web Flow. We chose Web Flow so the client could edit the website herself.

The client gave us a lot of creative freedom for the design. We decided on a horizontally scrolling landing page as a point of difference, with adding animations as the user scrolls. Other stand out features of the website include the scrolling animation timelines on the about and contact pages.
